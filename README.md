# padelore-site

Strona prawna/landing Padelore (App Store). Hostowane przez GitHub Pages.

- `/privacy.html` — Polityka prywatności (PL/EN) → wymagane przez App Store
- `/support.html` — Wsparcie + FAQ (PL/EN) → Support URL w App Store
- `/index.html` — landing

Custom domena `padelore.app` do podpięcia później (CNAME).
